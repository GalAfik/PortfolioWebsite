What tools do I use?

My Favorites

While I tend to use different tools for different jobs, the following tend to be my favorites for specific tasks. Rest assured that this is not a comprehensive list! Yes, I know how to use Microsoft Office, but who wants to hear about that?

Android Studio

I have used Android Studio for most of my application development on the Android system, though I prefer to do my game development using the Corona SDK Engine.

Arduino

I use Arduino boards for my hardware development and hackathon projects! I have built both a Camera glove and Mobility GPS system for the visually impared using Arduino hardware and it was some of my best work!

GitHub and BitBucket

I use BitBucket for my professional work on a dayily basis, although most of my personal work is versioned using GitHub. When it comes to version control, I'm kind of an "expert".

Bootstrap

I use either Twitter Bootstrap or Zurb Foundation for all of my weeb development work! In fact, this very site was made using Bootstrap. Isn't that neat?

Ultimaker Cura

Cura is 3D slicing software that I use to prepare my 3D sculpts for printing on my Ender 3 printer. It may be a bit loud, but I love it!

Drupal

While interning for Mindgrub in college, I learned to work with Drupal as a Content Management System and have used it both professionally during my time at Quotient Inc. and for any family friend who needs a new large-scale website!

GameMaker Studio

Ah, GameMaker. My very first game engine. I learned to use GameMaker in high school and was asked to teach my very first Game Design course that very same year! I have since taught countless courses in game design and development using GameMaker to students K-12!

GIMP

I use GIMP for much of my graphic design work. It's a lightweight alternative to Photoshop when I don't have my drawing tablet with me and it gets the job done quick!

JIRA

Atlassian's suite of tools, ranging from Confluence to JIRA is always a handy project management solution for the Scrum Master on the go! I even keep a Confluence page for my personal projects and notes at home!

JQuery

jQuery is a handy framework that I picked up during my stint as a front-end web designer at ECOMPEX and I've grown to love it since. It's a very handy tool for any web dev serious about their work!

Oracle Developer

From Oracle, to MySQL, to NoSQL, to PLSQL, I've worked on, around, and through databases the size of which you won't believe. Just ask me about my work with the Smithsonian Institute and Emammal sometime!

Photoshop

Photoshop was the first, and most powerful tool for graphic design and art that I learned. These days I use it to sketch and paint with my Huion Kamvas drawing tablet when I want to relax.

MS SQL Server

I use Microsoft's SQL Server Management Studio daily as a software engineer at Deutsch Bank. Database administration, while not glamorous, it always important!

Trello

I actually use Trello in my personal life much more than I do professionally! I love my lists. Please don't take my lists away. I need them.

Unity

Unity is currently my favorite game engine for small scale projects and low processing implementations! While not as robust as other game engines, Unity is very easy to learn and offers a great learning curve for my growing game design journey.

Unreal Engine

Unreal is a beast of a game engine. High resolution textures? Check. Virtual Reality integration? Check. This thing even has physics constraints! What more could I ask for in a game engine, except maybe some better Blueprint notes...

HTML, CSS, and JS

The 3 giants of web design. These handy languages have been with me since middle school and I would certainly be nowhere without them! While I try to keep up to date on all of the latest development frameworks, it's always nice to get back to some pure HTML and style things with CSS rather than SASS!

WordPress

While not my favorite Content Management System, I have to recommend WordPress for the ease of use and shallow learning curve. Anyone can use WordPress and that is exactly why I make all of my family's blogs with it. I just build out a basic site and release my aunt to do the rest! In no time, she has a beautiful site for her recipes and I can rest easy!

Recent Game Projects

Prepare to Re-Pair!

"Prepare to Re-Pair!" is the story of two lonely magnets, lost in the world and searching for love. Players will race against the clock to reach their electrically-charged buddies! Navigate mazes and solve puzzles to find your pair "re-pair" your friendship! This game was made using the Unity game engine in 48 hours during Global Game Jam 2020, hosted by Imangi Studios. The game can be played through the GGJ site at the link below. Check it out!
